Description

I created Retro Simon for my Q1 project at galvanize. I created this game in my fifth week of the program. In Q1 of the program we learned front-end technologies like Javascript, HTML and CSS. Simon is a front-end only game, because well that's all I knew at that point in the program. It uses Javascript and jQuery to manipulate the DOM and display the game effects.
